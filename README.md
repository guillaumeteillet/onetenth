# One-tenth

A Leiningen template for the PALO-IT TEAM.

## Usage

1째) Clone this repository : git clone https://github.com/guillaumeteillet/onetenth.git
2째) Create/Edit this file ~/.lein/profiles.clj and add this :

```
{:user {:plugins [[one-tenth/lein-template "0.1.2"]]}}
```

3째) Open a terminal, go to onetenth and run "lein install"
4째) Go to your desktop and you should be able to run "lein new one-tenth my-project"
